Dice game is a basic python program that runs in the terminal.

To run it you need to open the dice_game.py file and run it on a machine that has a python interpreter.

The players are asked questions and given prompts in the terminal. The game proceeds using player terminal input.

The base logic of the game is a player choice between rolling again or holding the value they have. If they roll a 1 the game ends their turn and passes to the next player.

If one of the players gets 100 points, instead of switching players, the game announces a winner.
