# anaplan dice game!

simple python dice game for anaplay interviewers to enjoy

## requirements
python3

# quickstart
python3 main.py

enter usernames as prompted
then answer 'y' or 'n' to reroll, any other input is equivelent to n
first to 100 or more wins
if you land on 1 your turn ends and you lose your unheld score
press ctrl + c to close anytime 
or n when the game ends
